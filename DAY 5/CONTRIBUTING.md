# Contributing

Thanks for your interest in contributing! This is a small project and contributions are welcome.

How to contribute
- Open an issue for bugs or feature requests.
- For code changes, fork the repo, make your changes on a branch, and open a pull request.
- Keep changes small and focused. Include a short description of what you changed and why.

Quick local test
- Open `index.html` in a browser or use Live Server in VS Code to preview changes.

Contact
- Add your name in the PR description or open an issue if you need guidance.
