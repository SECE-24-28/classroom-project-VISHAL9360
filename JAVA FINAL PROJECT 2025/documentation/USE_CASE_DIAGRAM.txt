╔══════════════════════════════════════════════╗
║          TECHNOLOGY STACK                    ║
╚══════════════════════════════════════════════╝

Frontend:
  - HTML5, CSS3, JavaScript
  - Responsive Web Design
  - Vanilla JS (No frameworks)

Backend:
  - Java 17
  - Spring Boot 3.1.0
  - Spring Data JPA
  - Spring Security
  - JWT Authentication

Database:
  - MySQL 8.0+
  - Hibernate ORM

Build Tool:
  - Maven 3.6+

╔══════════════════════════════════════════════╗
║          COMPONENT INTERACTION               ║
╚══════════════════════════════════════════════╝

1. USER REQUEST → Frontend sends HTTP request
2. CONTROLLER → Receives request, validates input
3. SERVICE → Business logic processing
4. REPOSITORY → Data persistence operations
5. DATABASE → Data stored/retrieved
6. RESPONSE → JSON data returned to frontend
7. UI UPDATE → Frontend updates DOM

╔══════════════════════════════════════════════╗
║          SECURITY LAYERS                     ║
╚══════════════════════════════════════════════╝

1. Authentication: Login/Register
2. Authorization: Role-based access (3 roles)
3. JWT Tokens: Session management
4. CORS: Cross-origin enabled
5. Security Config: Spring Security setup
